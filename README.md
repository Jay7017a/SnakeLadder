# SnakeLadder
Developed a console-based Snake and Ladder game using Core Java and the Collections Framework. The game supports two players, where each player rolls a virtual dice, moves across the board, climbs ladders, and slides down snakes according to the game rules. The winner is the first player to reach position 100.
The project uses HashMap to efficiently store snake and ladder positions, Random to simulate dice rolls, and Scanner for user input. It also demonstrates the use of loops, conditional statements, methods, and object-oriented programming concepts to manage the game flow.
Technologies Used
Core Java
Collections Framework (HashMap)
Random Class
Scanner Class
OOP Concepts
Key Features
Two-player game with custom player names.
Random dice generation (1–6).
Snake and ladder position handling using HashMap.
Displays each player's current position after every move.
Prevents movement beyond position 100.
Declares the winner when a player reaches exactly 100.
